Dr. B's Sweet Animation Skillz as Explained by Murry

Murry's book, Interactive Data Visualizaton for the Web, reduces the "sweet sweet animation", as described by Dr. B, to a two part procedure:

  1. Binding of the event "listener", that is awaiting for an event to happen to a specific DOM element.
  2. Giving instructions on what to do when the event happens.
Using svg, you can bind the element in the ususal manner. Use the on("click") [or hover or mouseover, etc. . . ] command to make a function occur. Define that function to create the effect you want. This can be a color change done on delay to make for a smooth color transition or between two graphs to have an animated transition between them.

Using the group function, it is possible to do many transitions at once to all grouped items.

An easily applied techinque that makes you look fancy!
